Detailed Description

지도도화면 위에 추가되는 Circle에 해당하는 Class.

지도화면 위에 Circle을 추가하기 위해서는 MTMapCircle 객체를 생성하여 MTMapView객체에 등록해 주어야 한다. (addCircle: (MTMapView)) Circle의 중심점을 설정하고 선 색상, 선 두께, 영역 색깔, 반경을 지정할 수 있다.
